# STEM by Sophia Website

Static website for stembysophia.com.

## Files
- `index.html` — homepage content
- `styles.css` — full visual styling
- `images/` — Sophia headshot and nature photography
- `CNAME` — GitHub Pages custom domain file

## GitHub Pages setup
1. Create a public GitHub repository, for example `stem-by-sophia`.
2. Upload all files/folders in this directory to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/root`, then save.
6. In the GitHub Pages custom domain box, enter `www.stembysophia.com`.
7. At your domain registrar, add a CNAME record: `www` → `YOUR-GITHUB-USERNAME.github.io`.
8. Once DNS propagates, enable **Enforce HTTPS**.

## Notes
The headshot is the original uploaded photo, not an AI-modified face. The improved visual background effect is handled by CSS framing around the image.
